Introduction

This project template allows for:

  1. Easy customisation of Bootstrap styles
  2. Auto page update on file content changes
  3. Type checking and a module system for Javascript via Typescript
  4. File concatentation via the Typescript compiler
  5. Source maps for easy debugging of style and scripts

The following panels describe how each of the above features work.

Visual Studio Code (VSC)

Visual Studio Code (VSC) provides a rich coding experience for client side assets.

Alt-Shift-F formats code and html.

The VSC task runner can run Gulp tasks and rerun those tasks when files change.

Alt-Shit-B runs the VSC default task

Alt-Shift-T runs the VSC test task

Gulp

Gulp is a task runner that can be called from the VSC task runner or from a CLI

Bootstrap
Various components are used

For example node, npm, bower, and less

Typescript

This project uses the TypeScript compiler which is set up to compile all of the *.ts files to Javascript and concatenate the *.js files into a single build.js file in the root folder.
